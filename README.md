# NoteApp

A simple Android note-taking application built with Kotlin.

## Team

- Kang Sokvimean
- Pen Panhaleap
- PHON KOEMSRAN

## Features

- **Create Notes**: Add new notes with title and content
- **Edit Notes**: Update existing notes
- **Delete Notes**: Remove notes with optional confirmation
- **User Profile**: View user info and logout
- **Settings**: Configure auto-save and delete confirmation

## Usage

1. **Login/Signup**: Create an account or login with existing credentials
2. **View Notes**: All your notes appear on the home screen
3. **Add Note**: Tap the + FAB button
4. **Edit Note**: Tap on any note to edit
5. **Delete Note**: Swipe or tap delete icon (confirmation required if enabled)
6. **Settings**: Access via bottom navigation gear icon to toggle auto-save and delete confirmation

## Build

```bash
./gradlew assembleDebug
```

## Requirements

- Android Studio
- Kotlin
- Gradle
